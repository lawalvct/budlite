üéØ Quick Answer: Where Are Salary Components Used?

Components appear on employee payslips AFTER you generate payroll. They show as individual line items in the Earnings and Deductions sections, with calculated amounts based on the employee's basic salary.

1

  • Click "Earnings" tab
  • Add: Housing, Transport, Bonus
  • Click "Deductions" tab
  • Add: Pension, Loan, Union Dues
2

Assign to Employees

  • Go to: Payroll ‚Üí Employees
  • Click "Edit" on employee
  • Scroll to "Salary Components"
  • Check components for this employee
  • Save changes
3

Create Period

  • Go to: Payroll ‚Üí Payroll Processing
  • Click "New Payroll Period"
  • Enter period dates
  • Save period
4

Generate Payroll

  • Open the period you created
  • Click "Generate Payroll" button
  • System calculates all components
  • Wait for completion
5

View Payslips

‚ö° Key Points to Remember

1. Components show ONLY AFTER generating payroll - not before
2. Each employee sees ONLY their assigned components - not all components
3. Amounts are calculated automatically - fixed amounts or percentages
4. Historical data preserved - old payslips don't change if you update components
